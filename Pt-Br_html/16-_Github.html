Github

Github

O QUE É GITHUB?

  • É um site, em que você pode concentra todo o seu trabalho de git em um só lugar, ele gerência o controle de versão feita pelo git. E também é reconhecido por ser uma rede social para os desenvolvedores.Porém todos podem usar independente da sua profissão. No github você consegue:

    • Ver trabalho de outras pessoas;
    • Trabalhar em projetos comunitários;
    • Versionar seu projeto, podendo deixar publico(todos podem ver) ou privado(só quem você permitir);
    • e outras coisas.

INICIANDO:

  • Crie uma conta no site: github.com;

  • Crie um repositório, o repositório seria o nome do projeto

  • Quando clicar novo repositório, você será direcionário para uma página em que contém informações como: nome do projeto, deixar público ou privado, adicionar readme e licensa, é só ler, optar por algumas configurações e seguir para a próxima passo.
  • Após fazer o comando , você será direcionado para essa página:

  • Você acaba de criar um git remoto, agora você tem o endereço http, do seu repositório remoto, para você vincular o git remoto com o git local, você precisa fazer alguns comandos.
  • Porém o github já ajuda nessa parte. Ele ensina com instruções, como mostrado na figura acima, copie e colar esses comandos no terminal.

INICIANDO O GIT REMOTO NO SEU PROJETO

  • Para deixar claro, qual comando faz para sincroniar o local com o remoto:
$ git remote add origin "http do github .git"

exemplo:

$ git remote add origin https://github.com/GuimaraesGabrielG/Tutorial-Git.git

Observe que o final é .git
