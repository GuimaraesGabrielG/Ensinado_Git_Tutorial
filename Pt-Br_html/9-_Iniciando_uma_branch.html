Iniciando uma Branch

Iniciando uma branch

CRIAR UM BRANCH :

$ git checkout -b "nome da branch"

exemplo:

$ git checkout -b funcionalidade1

Obs: Verifique que quando digitou esse comando ele fez uma cópia da raiz(master), assim é possível desenvolvar sua funcionalidade sem se preocupar com a master.

VOLTAR:

  • Para voltar para sua branch master, ou para outras branches que tenha criado:
$ git checkout "nome da branch"

exemplo:

$ git checkout master

LISTAR:

  • listar todas as branches:
$ git branch

REMOVER:

  • remover uma branch:
$ git branch -d "nome da branch"
