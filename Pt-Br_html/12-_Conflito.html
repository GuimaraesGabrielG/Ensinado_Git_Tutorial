Conflito

O QUE É CONFLITO?

Muitos desenvolvedores não sabem o que é conflito e como ocorre um conflito, e isso deixa o desenvolvedor em pânico, e talvez nem saiba como resolver o conflito.

  • O conflito ocorre quando dois desenvolvedores , pode ocorrer sozinho também, façam alterações distintas na mesma linha. Tornando impossível resolver o conflito sem intervenção humana. A segunda pessoa, entre os dois ou mais desenvolvedores, que enviar suas alterações, será informado do conflito, e escolherá se aplica sua alteração sobre a do outro, ou revisa a linha para manter ambas as alterações.

Você pode resolver o conflito:

  • Forma manual ➡ Abre o arquivo e apaga;
  • Vim;
  • Opendiff;
  • e outras maneiras;

Para chamar a ferramenta de interface gráfica

  • utiliza-se o comando:
$ git mergetool

Obs: Esse comando só irá funcionar quando o git reclamar de conflito

dependendo do tipo de configuração do git, ele abrirá a ferramenta.

  • Existem outras ferramentas para trabalhar com confito:
$ git mergetool --tool-help

$ git config — global merge.tool "nome da ferramenta"

Obs: Eu utilizo o "opendiff".
