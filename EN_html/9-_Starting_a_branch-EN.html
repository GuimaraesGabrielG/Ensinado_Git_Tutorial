Starting a Branch

CREATE A BRANCH : 

$ git checkout -b "branch name"

e.g:

$ git checkout -b functionality 1

Note: Check that when you typed this command it made a copy of the root (master), so it is possible to develop its functionality without worrying about the master.

COME BACK:

  • To go back to your master branch, or to other branches you have created:
$ git checkout "branch name"

example:

$ git checkout master

LIST:

  • list all branches:
$ git branch

TO REMOVE:

  • remove a branch:
$ git branch -d "branch name"
